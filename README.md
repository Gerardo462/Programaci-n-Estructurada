# Repositorio educativo – Programación Estructurada (GitHub Pages)

Este ZIP ya viene listo para:
- descomprimir
- subir a un repo nuevo
- activar GitHub Pages
- y que **NO marque 404** (rutas SIN acentos)

## Estructura
- `index.html` + `app.js` + `contenido.json` + `estilos.css`
- Carpetas de contenido:
  - `01_basico/`
  - `02_intermedio/`
  - `03_ejercicios/`
  - `04_pseudocodigo/`
  - `05_diagramas_flujo/`
  - `recursos/`

## Pasos para publicar
1. Crea un repo nuevo en GitHub (por ejemplo: `Repositorio-programacion-estructurada`)
2. Sube **todo** el contenido de este ZIP a la raíz del repo
3. Ve a **Settings → Pages**
4. En *Build and deployment*: selecciona **Deploy from a branch**
5. Branch: `main` y carpeta `/ (root)` → Save
6. Espera a que GitHub Pages genere tu link

## Importante (para que se vea tu contenido real)
Los archivos `.md` y `.py` vienen como **plantilla**.  
Solo abre cada uno y pega tu contenido real (clase, ejercicios, etc.).
