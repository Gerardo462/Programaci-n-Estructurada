# Guía básica de instalación para programar

Antes de comenzar a programar, es necesario preparar el entorno de trabajo.  
Esto significa instalar algunas herramientas que permiten **escribir y ejecutar código en la computadora**.

Esta guía explica de forma general **cómo instalar y ejecutar programas en cualquier lenguaje de programación**.

---

## ¿Qué se necesita para programar?

Para empezar a programar normalmente se necesitan dos cosas:

1. **Un editor de código**
2. **El lenguaje de programación**

### Editor de código

Un editor de código es un programa que permite escribir y organizar código.

Algunos editores populares son:

- Visual Studio Code
- Sublime Text
- Notepad++
- Atom

Uno de los más recomendados para principiantes es **Visual Studio Code**.

---

## Instalar Visual Studio Code

Visual Studio Code es un editor gratuito muy utilizado por programadores.

### Paso 1

Ir al sitio oficial:

https://code.visualstudio.com

### Paso 2

Descargar la versión correspondiente a tu sistema operativo:

- Windows
- Mac
- Linux

### Paso 3

Ejecutar el instalador y seguir las instrucciones.

Después de instalarlo podrás **crear y editar archivos de código**.

---

## Instalar un lenguaje de programación

Cada lenguaje necesita su propio **intérprete o compilador**.

Por ejemplo:

| Lenguaje | Qué se instala |
|--------|----------------|
| Python | Intérprete de Python |
| Java | JDK (Java Development Kit) |
| C | Compilador GCC |
| JavaScript | Node.js |

---

## Ejemplo: instalar Python

Python es uno de los lenguajes más fáciles para empezar.

### Paso 1

Ir al sitio oficial:

https://python.org

### Paso 2

Descargar la versión más reciente.

### Paso 3

Durante la instalación **activar la opción**:

Add Python to PATH

Esto permite ejecutar Python desde la terminal.

---

## Verificar la instalación

Después de instalar un lenguaje, es importante comprobar que funciona.

Abre una **terminal o consola** y escribe:

```bash
python --version
```

Si la instalación fue correcta, aparecerá algo como:

```
Python 3.12.1
```

---

## Crear tu primer programa

Abre tu editor de código y crea un archivo.

Ejemplo para Python:

```
hola.py
```

Dentro del archivo escribe:

```python
print("Hola mundo")
```

Guarda el archivo.

---

## Ejecutar el programa

Abre la terminal en la carpeta donde guardaste el archivo y escribe:

```bash
python hola.py
```

La salida será:

```
Hola mundo
```

---

## Flujo básico para programar

El proceso general para programar es:

1. escribir el código en un editor
2. guardar el archivo
3. ejecutar el programa
4. observar el resultado
5. corregir errores si es necesario

---

## Recomendación

Si estás comenzando a programar, se recomienda empezar con:

- Python
- JavaScript
- C

Estos lenguajes tienen mucha documentación y recursos educativos.
