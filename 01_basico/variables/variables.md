
## ¿Qué es una variable?

Una **variable** es un espacio donde un programa puede guardar información para usarla después.

Podemos imaginar una variable como una **caja con nombre** donde guardamos un valor.

Por ejemplo:

```python
edad = 20
```

Aquí la variable **edad** guarda el número **20**.

Las variables permiten:

* guardar información
* hacer cálculos
* usar resultados después
* controlar el comportamiento del programa

---

## Ejemplo sencillo

Observa el siguiente código:

```python
a = 5
b = 3
suma = a + b
```

En este caso:

* la variable **a** guarda el número **5**
* la variable **b** guarda el número **3**
* la variable **suma** guarda el resultado de **5 + 3**

Entonces el valor de **suma** será **8**.

## Cambiar el valor de una variable

Las variables **pueden cambiar su valor durante la ejecución del programa**.

Por ejemplo:

```python
numero = 7
numero = numero + 3
```

Primero la variable **numero vale 7**.

Después el programa suma **3**, por lo que ahora **numero vale 10**.

Esto se usa cuando un programa necesita **actualizar información**.

## Tipos de información que puede guardar una variable

Una variable puede guardar diferentes tipos de datos.

### Números

```python
edad = 21
temperatura = 18
```

### Texto

```python
nombre = "Ana"
ciudad = "México"
```

### Resultados de operaciones

```python
x = 4
y = 2
resultado = x * y
```

En este caso **resultado** guardará **8**.
