
## ¿Qué es una variable?

Una **variable** es un espacio donde un programa puede guardar información para usarla más tarde.

Podemos imaginar una variable como **una caja con nombre** donde guardamos un valor.

Por ejemplo, si tenemos una caja llamada `x` y guardamos el número 10 dentro, decimos que:


x = 10


Esto significa que la variable `x` guarda el valor **10**.

---

## ¿Para qué sirven las variables?

Las variables permiten:

- guardar información
- hacer cálculos
- usar resultados después
- controlar el comportamiento de un programa

Casi todos los programas usan variables para trabajar con datos.

---

En este caso:
